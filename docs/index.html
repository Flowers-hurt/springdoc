基于 JavaDoc

利用 JavaDoc 注释自动生成文档，无需额外注解，零侵入式开发体验。让你的代码保持整洁，专注于业务逻辑而非文档标记

📋

OpenAPI 规范

自动生成符合 OpenAPI 规范的结构体，保证文档的标准化和可移植性。支持 Swagger 相关工具链的所有功能

🔍

Knife4j 集成

内置集成 Knife4j 作为接口文档的可视化界面，提供美观易用的浏览体验。支持接口调试、文档导出等高级功能

🚀

简单易用

只需添加依赖并简单配置即可使用，支持 SpringBoot 2.x 和 3.x 版本，兼容 Java 8 至 Java 17

🔄

实时同步

修改代码注释后，文档自动更新，确保代码与文档始终保持一致，避免文档过时问题

🛠️

高度可定制

提供丰富的配置选项，可自定义文档标题、描述、版本号、服务条款等信息，满足各种项目需求
